# Product Trust Box for WooCommerce

Adds a customizable **Trust Box** to the WooCommerce single product page.

## Features

- Displays a trust box on WooCommerce single product pages
- Option to place the box above or below the Add to Cart button
- Easy customization from the WordPress admin panel

## Installation

1. Upload the plugin folder to `/wp-content/plugins/`
2. Go to **WordPress Admin → Plugins**
3. Activate **WooCommerce Product Trust Box**

## Usage

Once activated, the trust box will automatically appear on the single product page.

To customize the content go to:

WooCommerce → Product Trust Box (Settings)

## Requirements

- WordPress
- WooCommerce
