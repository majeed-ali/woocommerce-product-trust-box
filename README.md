#WooCommerce Product Trust Box

Adds a customizable Trust Box to the WooCommerce single product page, displayed above or below the Add to Cart button. This box is typically used to show trust signals such as guarantees, shipping info, security badges, or other reassuring messages to customers.

Features

Displays a trust box on WooCommerce single product pages

Option to place the box above or below the Add to Cart button

Easy customization from the WordPress admin panel

Perfect for showing:

Secure checkout messages

Shipping information

Money-back guarantees

Trust badges

Store benefits

Installation

Download or clone this repository.

Upload the plugin folder to:

/wp-content/plugins/

Go to WordPress Admin → Plugins.

Activate WooCommerce Product Trust Box.

Usage

Once activated, the trust box will automatically appear on the single product page.

You can control and customize the content from:

WooCommerce → Product Trust Box (Settings)

From there you can:

Edit the trust box content

Change what text or HTML appears

Adjust how the message is displayed

Where It Appears

The trust box appears on the WooCommerce Single Product Page, either:

Above the Add to Cart button, or

Below the Add to Cart button

(depending on plugin configuration).

Customization

To edit the trust box:

Go to WooCommerce → Product Trust Box (Settings)

Modify the content fields

Save changes

The updates will instantly apply to all product pages.

Requirements

WordPress

WooCommerce installed and activated
