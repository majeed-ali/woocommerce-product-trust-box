Adds a configurable trust / USP box on WooCommerce single product pages. The trust box can display delivery information, free shipping notices, discounts, and secure payment icons to improve customer confidence before purchasing.

Features include:
- Delivery date range (today + X days to today + Y days)
- Free delivery text with optional minimum order value
- Discount line
- Secure payment line with selectable payment icons
- Optional uploaded PNG or SVG payment icons
- Choose display position (above short description, below short description, below add to cart button, or after add to cart form)

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/` or upload the ZIP via WordPress admin.
2. Activate **Sirpi Trust Box for WooCommerce**.
3. Go to **WooCommerce → Sirpi Trust Box** to configure the settings.

== Frequently Asked Questions ==

= Where does the trust box appear? =

The trust box appears on WooCommerce single product pages. The position can be configured in the plugin settings.

= Do I need WooCommerce installed? =

Yes. This plugin requires WooCommerce to be installed and activated.
