Adds a configurable trust/USP box on WooCommerce single product pages:
- Delivery date range (today + X days to today + Y days)
- Free delivery text with optional minimum order value
- Discount line
- Secure payment line with selectable payment icons and optional uploaded PNG/SVG icons
- Choose where to display (above short desc, below short desc, below add-to-cart, after add-to-cart form)

== Installation ==
1. Upload the plugin folder to /wp-content/plugins/ or upload the ZIP in WP Admin.
2. Activate "WooCommerce Product Trust Box".
3. Go to WooCommerce → Product Trust Box to configure.
